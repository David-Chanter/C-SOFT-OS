About

OS is a fast, reliable Graphical User Interface Operating Environment for the Commodore 64. It is free to download, and it comes with several multi-purpose applications built in for Business or Personal use. It is written in BASIC, so it can be edited with ease. If you do not wish to edit it, it will work as a fully-functional operating system. Originally, I had created a Word Processor and I wanted to give it a fancy loader or launcher like some old games and software form the 1980s. Once I had designed it, I thought "Why not turn this into an operating system?", and that I did.

Requirements

In order to get the best OS experience, you will need to have a Commodore 64, 64C, 64G, SX-64, 128 (in 64 mode), an emulator (eg. VICE), THEC64, THEC64 Mini, or THEVIC20. If you have more than one of these platforms, you should use the best one. You will also need to have one of the following storage devices: Commodore disk drive, SD2IEC, Pi1541, or a USB stick (For THEC64 line of computers).
